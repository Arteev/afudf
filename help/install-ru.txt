Пример установки для Windows:
 1. Установите FireBird (рекомендуется версию выше 1.5)
 2. Скопируйте файлы библиотек dll в папку {FireBird}\UDF 
    (обычно Program Files\FireBird\FireBird_1_5\UDF)
 3. Скопируйте iconv.dll в системную папку windows\system32
 3.1 Скопируйте afmmngr.dll в системную папку windows\system32 
 4. Подключиться к базе данных 
 5. Для используемой базы данных зарегистрировать функции выполнив SQL скрипты регистрации
*6. Возможно нужно выполнить chgtowin.sql для исправления имен библиотек.После выполнения данного 
    скрипта рекомендуется перезапустить сервер БД
 
Пример установки для Linux:
 1. Установите FireBird (рекомендуется версию выше 1.5)
 2. Скопируйте файлы библиотек so в папку {FireBird}/UDF 
    (обычно /opt/firebird/UDF) 
 3. настройте ldconfig чтобы искал библиотеки {FireBird}/UDF 
 4. Подключиться к базе данных
 5. Для используемой базы данных зарегистрировать функции выполнив SQL скрипты регистрации
*6. Возможно нужно выполнить chgtolinux.sql для исправления имен библиотек.После выполнения данного 
    скрипта рекомендуется перезапустить сервер БД

 
FireBird 2.x Внимание! Каталоги в которые библиотеки скопированы должны быть доступны для firebird в параметре
 UdfAccess, либо это параметр должен быть равен Full
